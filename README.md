Message Validation with Callback
A simple JavaScript example that checks user input for a forbidden character (#) using a callback function.
How it works
Trims extra spaces from the input text
Checks if # exists in the message
If found, executes a callback and clears the message
Purpose
Practice string methods (trim, split, indexOf, replace)
Understand how callback functions work in a real scenario

(practice2)
User Authentication with Callback
A simple JavaScript example that validates a username and password using callback functions.
How it works
Trims extra spaces from the username
Checks if the username is "sara" and password is 123
Executes corecct() callback if credentials are correct
Executes incorecct() callback if credentials are incorrect
Purpose
Practice input validation and string methods (trim)
Understand how callback functions work in a real scenario
